# tuple-pak-saikin_Gilang-2
Praktikum 
Dictionary (dict) → menyimpan data per hari
List → menampung banyak item
Tuple → menyimpan data barang (nama, jumlah, harga)
Key (dict) = hari (senin, selasa, rabu)
Value = list berisi tuple

Isi tuple:

(nama_barang, jumlah, harga)
Menggabungkan semua data dari berbagai hari
Menghitung:
total jumlah barang
total harga per barang
Mencari barang berdasarkan kata kunci (keyword)
Mengurutkan barang berdasarkan jumlah terbanyak
Menampilkan:
Hari
Nama barang
Jumlah
Harga
Menampilkan total belanja setiap hari
Menggunakan:

sum() untuk menjumlahkan semua hari
Menampilkan:

Total jumlah tiap barang
Total uang yang dikeluarkan per barang
Menampilkan:

Barang yang paling sering dibeli
User bisa mencari:

nama barang tertentu
Logika:

Jika total > 100000 → diskon 10%
Jika tidak → tidak ada diskon
